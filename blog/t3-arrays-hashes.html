Astonishingly, I have learned a full set of index flashcards full of terms that would have been Greek to me just a few weeks ago, but now I'm like "THIS IS SPARTA!". They include bash commands, Ruby methods, Git commands, and many HTML and CSS attributes and tags.

I'm feeling up to the task of attempting to explain the main difference between an array and hashes in Ruby.

From what I have gathered, in my extremely limited time as a student of the subject, an array consists of an ordered group of objects, (indexed by sequential integers,) as opposed to a hash which can be assigned any old index imaginable, numbers, strings, different types of fruit. The main thing is that hashes are not ordered, and arrays are. I hope this is correct, otherwise I'll look back on this in six months and wonder just what was I thinking?
